# R Journal Paper

This repository contains the R Markdown file for the illustration section of the
paper.

## File to Edit

Please edit only this file:

```text
illustration_section.Rmd
```

## How to View the Rendered Paper

Open `illustration_section.Rmd` in RStudio and click **Knit**.

RStudio will render the R Markdown file and show the paper output.

## How to Suggest Edits from RStudio

Please make changes in RStudio (preferred), then submit the changes to GitHub.

1. Clone this repository from GitHub.
2. Open `R_Journal_Paper.Rproj` in RStudio.
3. Open `illustration_section.Rmd`.
4. Click **Knit** in RStudio to view the rendered paper.
5. Edit the R Markdown file in RStudio.
6. Commit the changes to `illustration_section.Rmd` with a short commit message.
7. Push the commit to a separate branch on GitHub.
8. Open a pull request from that branch.
